# Análise do questionário sobre pesticidas em Granja de Pessubé

Este repositório organiza os dados, scripts, gráficos e aplicaçăo web relacionados com a análise do questionário aplicado ŕs horticultoras de Granja de Pessubé sobre práticas de uso de pesticidas, conhecimentos de segurança e perceçőes sobre impactos na saúde humana e no ambiente.

## Objetivos da análise

- Identificar práticas hortícolas associadas ao uso de pesticidas.
- Averiguar conhecimentos sobre medidas de segurança no manuseio, aplicaçăo, armazenamento e descarte de pesticidas.
- Compreender os impactos percebidos pelas horticultoras na saúde e no ambiente.
- Analisar associaçőes entre escolaridade, idade, experięncia, formaçăo e perceçőes.
- Apoiar a proposta de estratégias de educaçăo ambiental.

## Estrutura do repositório

```text
data/
  raw/              Dados originais năo modificados. Năo recomendado subir dados sensíveis.
  processed/        Dados limpos, anonimizados e tabelas de análise.
  app/              Dados exportados para uso na aplicaçăo web.

scripts/            Scripts Python para limpeza, análise e exportaçăo.
notebooks/          Notebooks de exploraçăo e validaçăo.
images/             Gráficos exportados para relatório, tese ou apresentaçăo.
docs/               Aplicaçăo web publicada via GitHub Pages.
```
